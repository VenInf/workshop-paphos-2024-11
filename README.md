# Workshop on Datatypes

A workshop on functional programming and datatype prepared for JetBrains.

Topics to cover:
* Basic Haskell
* Algebraic data types
* Typeclasses (Functor, Monoid, Foldable)
* Functor fixpoints
* [Recursion schemes][0]
